NUESTROS SERVICIOS

MONITOREO DEL CULTIVO

Se trata de una aplicación móvil SIMCA, la cual será manipulada por el encargado de cada finca, esta permite el registro de plagas, enfermedades, producción, fumigación, plantación y demás actividades propias del agro, teniendo así una trazabilidad completa de cada lote y/o cama que integre la finca.

CONTROL DEL INVENTARIO

Permite el ingreso de múltiples productos (herramientas, insumos, materia prima), así mismo ofrece la opción de extracción y adición (total o parcial) de los productos previamente ingresados; Dicho modulo posee la característica de registrar todos los movimientos que se realizan, conociendo detalles exactos tales como fecha, hora, cantidades, precios, entre otros.

CONTROL DE HORAS TRABAJADAS

Sistema que utiliza un lector RFID para identifica al empleado por medio de una tarjeta inteligente que se sobrepone en el dispositivo, inmediatamente la fecha y hora es detectada, para luego almacenarla y de esta manera poder hacer el cálculo de las horas trabajadas durante el día
